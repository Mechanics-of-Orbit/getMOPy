MOPy | Learn Orbital Mechanics

Mechanics of Orbit Using Python(MOPy) is a user-friendly tool that can be operated with ease by the user who has very limited knowledge about the concepts of orbital mechanics. It offers users to learn about a particular concept with a brief explanation so that the user can gain the theoretical knowledge required through visualizations. Educators can use this as a teaching tool directly as it has been developed module wise so they can use the corresponding moduleâ€™s tool. The 3D environment benefits the learners to visualize the fundamental concepts easily. It assists the user to verify manually calculated data by serving as an advanced virtual calculator. MOPy is an open source software with GNU-GPL v3 license which anyone can use or work with. It runs on the Windows platform only at present.
